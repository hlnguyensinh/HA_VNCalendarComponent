# VN Lunar Calendar Component Services

## Details:

| Service            | Description                               |
| ------------------ | ----------------------------------------- |
| `today`            | Get today's lunar information             |
| `get_day`          | Get lunar info of a specific day          |
| `get_month`        | Get full month lunar data                 |
| `get_year`         | Get full year lunar data                  |
| `good_hours`       | Get good hours in a day with current time |
| `good_hours_today` | Get good hours today with current time    |
